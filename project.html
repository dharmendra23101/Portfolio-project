SUDOKU SOLVER

Developed a dynamic Sudoku Solver web application using HTML, CSS, and JavaScript. The application features:

  • Interactive Sudoku Grid: Users can input values with real-time validation to prevent incorrect entries.
  • Automated Solver: Implements a backtracking algorithm to solve Sudoku puzzles efficiently.
  • Responsive Design: Ensures a smooth user experience across different screen sizes.
  • Error Handling: Highlights invalid inputs and alerts users about incorrect entries.
  • Clear Grid Functionality: Allows users to reset the puzzle with a single click.
